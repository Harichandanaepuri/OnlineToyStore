Project Setup:
Create "test" database in mongodb and the add the collections using json files. Start the mongo server.
Go to the project folder (OnlineToyStore) and install npm dependencies using npm i
Go to OnlineToyStore/frontend folder and install dependencies using npm i
Start the backend server from command prompt as follows:
	Go to OnlineToyStore and type the following command "set port=3001 && node bin/www". The backend server runs on port 3001.
	Start the application by switching to folder (OnlineToyStore/frontend) and type the following command "npm start". The application starts on port 3000.
